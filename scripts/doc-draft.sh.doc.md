# doc-draft.sh

## What this file does
Draft a section in an existing session, with clarifying questions. Equivalent
to `sin-doc draft` or the MCP `doc_section_draft` tool.

## Usage examples
```bash
./scripts/doc-draft.sh abc12345 "Installation"
```
