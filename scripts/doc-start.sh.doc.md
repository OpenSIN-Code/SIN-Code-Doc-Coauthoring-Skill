# doc-start.sh

## What this file does
Start a new doc coauthoring session from the command line. Equivalent to
`sin-doc start` or the MCP `doc_start` tool.

## Which other files import / touch it
- None (called by humans)

## Usage examples
```bash
./scripts/doc-start.sh README "My Project"
./scripts/doc-start.sh ADR "Use PostgreSQL" "" "Decide primary store"
```
