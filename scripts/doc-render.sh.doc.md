# doc-render.sh

## What this file does
Render a draft to a final format. Equivalent to `sin-doc render` or
the MCP `doc_format_render` tool.

## Usage examples
```bash
./scripts/doc-render.sh abc12345
./scripts/doc-render.sh abc12345 html
./scripts/doc-render.sh abc12345 pdf
```
