# install.sh

## What this file does
One-line installer for `sin-doc-coauthoring`. Creates a `.venv`, installs
the package with dev + pdf extras, and (optionally) installs the bash
wrappers to `~/.local/bin/`.

## Usage examples
```bash
./install.sh
```
